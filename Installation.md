# ⚙️ Instalação

## 📦 Requisitos

* Spigot 1.8.8
* Java 8
* PlaceholderAPI (opcional)

---

## 📥 Instalação

1. Coloque o `.jar` em:

```
/plugins/
```

2. Inicie o servidor

3. Configure o arquivo:

```
/plugins/MixedMines/config.yml
```

4. Reinicie ou use:

```
/mixedmines reload
```

---

## ✅ Verificação

Use:

```
/mixedmines info
```

Se não der erro, está funcionando.

---

## 🔌 PlaceholderAPI

Se estiver usando:

```
/papi ecloud download mixedmines
/papi reload
```
