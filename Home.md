# ⛏️ MixedMines Wiki

Bem-vindo à documentação oficial do **MixedMines**.

Plugin avançado de minas para servidores **Spigot 1.8.8**, focado em performance, flexibilidade e experiência profissional.

---

## 🚀 Principais Recursos

* Sistema de minas configuráveis
* Reset automático por porcentagem e tempo
* Sistema de bônus completo
* Ranking de mineradores
* Integração com PlaceholderAPI
* Efeitos visuais (Title, ActionBar, Firework)
* Alta performance para servidores 1.8.8
